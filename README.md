## 実行に必要なもの

### 必須ツール

* **Python** 3.10 以上推奨
    
* **FFmpeg**
    
    * `brew install ffmpeg`
        

### 必須Pythonパッケージ

* `ffmpeg-python`（録音に使用）
    
* `pydub`（音声の読み込み・変換）
    
* `numpy`
    
* `mlx-whisper`（Apple Silicon で軽快に動く Whipser 実装）
